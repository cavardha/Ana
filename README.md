# Ana's Cleaning Services — Professional Website

This is a complete VS Code-ready website for **Ana's Cleaning Services** in Dallas, TX.

## Included

- `index.html` — main website page
- `css/style.css` — full desktop and mobile styling
- `js/main.js` — mobile menu, before/after sliders, WhatsApp quote form, animations
- `assets/images/` — all images used in the website
- `docs/` — client notes, launch checklist, and photo replacement guide

## How to open in VS Code

1. Extract the ZIP file.
2. Open VS Code.
3. Click **File → Open Folder**.
4. Select the folder `ana-cleaning-dallas-pro`.
5. Install the VS Code extension **Live Server**.
6. Right-click `index.html` and choose **Open with Live Server**.

## Important before publishing

The website uses cropped brand assets from the Instagram screenshots you provided. The before/after images are demo website visuals. Before final publishing for the client, replace demo before/after photos with real cleaning-job photos from Ana's team.
